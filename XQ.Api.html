XQ v-1.0.1 API---需要jq库
1、XQ.autoWidth(a,b)
    据子集自动宽度a是父级，b是子集

2、XQ.WinW 
    窗体宽度

3、XQ.WinH 
    窗体高度

4、XQ.autoWinW（a）
    自动为窗口宽度 ~a是元素

5、XQ.XQ.autoWinH （a）
       自动为窗口高度~ a是元素

6、XQ.goTop（a,b）
    滚动到设定的距离显示回到顶部按钮，点击回到顶部~a是元素，b是距离比

7、XQ.scrollShow（a,b）
    滚动到某个文职显示元素~a是元素，b是距离比；

8、XQ.scrollFixed(a,b,c)
    滚动距离来判定添加减少类型~a是元素，b是距离；c是变换的类名

9、XQ.scrollTop
    获取滚动条高度

10、XQ.banFade (a,b,c,d)
    banner渐隐渐现 ~ a是父级；b是子集；c是间隔，d是从几个开始（仅仅是js不包含样式）;

11、XQ.setCookie (a,b,c,d,)
    简单的写入cookie ~a是名称；b是值，c是存活时间,d是时间类型（day）

12、XQ.getCookie（a）
    简单获取cookie~a是名称;

13、XQ.removeCookie（a）
    简单移除cookie~a是名称

14、XQ.banSwipt(a, b, c, d, dd, e)
    简单的banner向上移动~a=运行的父级，b=运行标签，c=间隔时间（ms），d=父级高度,dd=滚动误差,e=回调函数

15、XQ.clickShow (a, b, c, d)
    点击显示隐藏~a是点击显示按钮，b是点击隐藏按钮，c是执行元素(数组形式)，d是回调函数

16、XQ.moborpc()
    判断是否是移动端：true是移动端；

17、XQ.timeAuto(a,b);
    延时执行，并且重复执行，时间大于3000秒后停止执行~a是回调函数，b是时间间隔默认为300；

18、XQ.download（a）
    当所有资源都OK后执行；~a是回调函数；

19、XQ.slideItem （obj, options）
    banner滚动一=>obj是滚动元素;options是配置文档内有函数：
            autoplay: false, //是否自动播放
            times: 3000, //自动播放间隔时间
            hoverstop: true, //鼠标悬浮停留
            flag: null, //
            container: null, //slide容器的类名
            slide: null, //滚动的类名
            item: null, //子集的类名
            itemindex:3,//滚动数量
            media: false, //是否在什么尺寸进行地洞滚动；
            min_width: 0, //滚动失效的最小尺寸；
            next: null, //下一个按钮的类名
            prev: null //前一个按钮的类名

20、XQ.switchOver （obj, options）
        点击切换列表~obj是一个字符串，多个元素以逗号“,options是配置项：
                classs: 'dq', //切换当前的选中状态的class
                switchs: false, //是否切换相应的内容
                obj2: false, //切换的内容/可以是其父级的类、id，按照索引值进行切换，或者是数组(类名或id组成)，按照索引值对应数组进行切换
                call: function() {} //你懂得...

21、XQ.popup （option）
        弹出框~options是配置项：
            obj: '', //点击
            popup_zc: '', //弹出遮层
            popup: '', //弹出框
            closes: '', //关闭按钮
            mobile: true, //手机上执行
            ways: 'fade', //出现方式可以是fade可以是左右上下
            num: 0, //若是左右上下那就输入相应数值
            showbefore: function() {
                return true
            }, //打开前执行...
            showafter: function() {}, //打开后执行...
            closebefore: function() {
                return true
            }, //关闭后执行...
            closeafter: function() {}, //关闭后执行...
            call: function() {}

22、XQ.trim（a）
        去除空格 ~a是字符串

23、XQ.placeholer()

 4、 $().fullimg ()
        图片缩放，拉伸；配置：
                classs: 'fullimg', //图片新建容器的类名

                    return false
                }

            $('.modul-banner-2 .banner-ul-2 ul li .img img').fullimg({
                    resize: true,
                    fullScreen: true
                })

10、   $().parallax()
        视滚动差效果
             ratio: 0.5, //移动比例
            direct: 'down', //移动方向-可选up/down/left/right/leftdown/leftup/rightup/rightdown

        例子：

        ratio:0,
        max_top:600,
        addclass:'bk-nav'
        })
